# Sales Data Processing & Analytics using PySpark

## Overview
This project demonstrates an end-to-end ETL pipeline built using PySpark to
process and analyze sales data for meaningful business insights.

## Technologies Used
- Python
- PySpark
- Apache Spark
- Git & GitHub

## Project Workflow
- Ingested raw sales data from CSV files
- Cleaned and transformed the data
- Performed aggregations and analytics
- Generated insights for decision-making

## How to Run
1. Install dependencies:
   pip install pyspark
2. Run the script:
   python scripts/etl_pipeline.py# Sales-Data-Processing-Analytics-using-PySpark-
